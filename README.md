<h4>IncludeJS Build Tool For node.js</h4>

<p>@see <a href='https://github.com/tenbits/IncludeJS'>IncludeJS Project</a>

<p>
<div>Install:</div>
<code>$ npm install includejs -g</code>
</p>
<p>
<div>Help:</div>
<code>$ includejs -help</code>
</p>
<p>
<div>Example:</div>
<code>cd ../project <br/>
includejs index.html -minify</code>

</p>
